字体属性

​ css字体属性定义字体，颜色，大小，加粗，文字样式。

1.color

  • 建议使用十六进制表示颜色，#+改颜色的十六进制
  • rgba中的a是alpha，代表透明度，0~1。

2.font-size

表示字体大小

  • 注意chrome浏览器最小字体是12px。

3.font-weight

  • bold定义粗体字符
  • bolder定义更粗的字符
  • lighter定义更细的字符
  • 100-900定义由细到粗，400等同默认，700等同于bold

4.font-style

指定字体样式

  • font-style：normal
  • font-style：italic

5.font-family

指定一个元素的字体：例如微软雅黑等

块级标签和行内标签

发表于 2022-08-12
本文字数： 426 阅读时长 ≈ 1 分钟

块级元素

  • 它们自己独占一行，无论是否换行标签< br>，块级元素总是再新行上开始的
  • 块级元素的宽度width、高度height以及外边距和内边距等都是可以控制的
  • 如果省略块级元素的宽高，那么它的默认宽高为当前浏览器的宽高
  • 块级元素中可以包含其它的内联元素和块级元素

代表性的元素有< div>,以及p、nav、aside、header、footer、section、article、ul、address、h1~h6等。

内联元素

内联元素也称为行内元素，行内元素最常用的是span，以及b、i、u、em、strong、sup、sub、big、small、li、ins、del、code、cite、dfn、var等。

  • 行内元素和其他元素会在同一行显示
  • 行内元素的宽高以及外边距和内边距都不可改变
  • 行内元素的宽度就是其中内容的宽度，不可改变
  • 内联元素中只能容纳文本或者其它内联元素

行内块级元素

特点：不换行可以识别宽高，例如button、img、input等。

css选择器

发表于 2022-08-12
本文字数： 3.3k 阅读时长 ≈ 3 分钟

CSS选择器

1.通用选择器

通用选择器(*)选择页面上的所有HTML元素。

2.元素选择器

元素选择器根据元素名称来选择HTML元素。

​ 这里我发现一个问题，span标签如果使用text-align属性会无效，如果想使用改属性可以加一个p标签嵌套。原因如下：

  • 如果想text-align起作用的化，需要满足作用元素是一个行内元素
  • 选择器要指向其父元素（父元素必须为块元素）

3.类选择器

类选择器选择特定class属性的HTML元素。

如果指定选取某个class的元素，在（.）+类名即可。

当一个元素有多个类名的时候，从上往下执行，最后的属性将覆盖之前的属性。

4.id选择器

id选择器使用HTML元素的id属性来选择指定元素。

元素的id在页面中是唯一的，所以id选择器用于选择一个唯一的元素！

使用方法#+该元素的id。

发现类名不能纯数字！注意！

5.分组选择器

分组选择器选取所用相同样式定义的HTML元素。

选择器优先权

行内样式>id选择器>类选择器>元素选择器>通用选择器

HTML5新增标签

标题标签

发表于 2022-08-11
本文字数： 919 阅读时长 ≈ 1 分钟

标题通过< h1> - < h6> 进行定义的

其中 < h1> 定义最大的标题， < h6> 定义最小的标题

生成h1~h6快捷键：h$*6

需要正确使用标题

  • 确保将HTML标题标签只用于标题，而不是加粗等其它功能
  • 正确使用标题有益于SEO
  • 应该将 < h1> 用作最重要的标题，其次是 < h2> ,以此类推
